# CV Repository

A professional, responsive CV website built with HTML, CSS, and JavaScript.

## Features

- 📱 Fully responsive design
- 🖨️ Print-friendly and PDF-ready
- 📝 Easy-to-edit JSON-based content
- ⚡ GitHub Pages deployment
- 🎨 Modern, professional styling

## Quick Start

1. Edit `cv.json` with your personal information
2. Add your profile photo to `assets/profile.jpg`
3. Enable GitHub Pages in repository settings
4. Your CV will be live at `https://fharisite.github.io/cv/`

## File Structure

- `index.html` - Main CV webpage
- `style.css` - Responsive styling
- `script.js` - Dynamic content loading
- `cv.json` - Your CV content (edit this!)
- `.github/workflows/deploy.yml` - Auto-deployment configuration

## Customization

Edit the `cv.json` file to update your:
- Personal information
- Work experience
- Education
- Skills
- Projects
- Certifications

All changes will be automatically reflected on your CV website!
